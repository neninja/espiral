## Desenvolvimento

### Build local

- Todos formatos
```sh
make all
```

- Livros (PDF e EPUB)
```sh
make book
```

- Site
```sh
make html
```

> Caso não possua docker e/ou make, dê uma olhada no funcionamento do `Makefile`, `docker-compose-*.yml` e seus scripts em bash para ver quais dependências baixar para executar
